# UE5 Gameplay Systems (C++)

This repository showcases gameplay systems developed in Unreal Engine 5 using C++.

## Systems Included
- Player Character (movement, input, combat)
- Enemy System (base class, damage handling)
- Projectile System
- Kill Volume and respawn logic

## Technical Focus
- Gameplay system architecture
- C++ implementation in Unreal Engine 5
- Real-time debugging and iteration

## Additional Experience
In addition to this C++ work, I have developed a full gameplay vertical slice using Unreal Engine 5 Blueprints, including AI behavior, wave-based encounters, and game state systems.
