###### \# UE5 Gameplay Systems (C++)

###### 

###### This repository contains gameplay systems developed in Unreal Engine 5 using C++.

###### 

###### \## Overview

###### This project focuses on implementing core gameplay mechanics and systems, including player movement, combat interactions, and enemy behavior. The goal is to demonstrate practical C++ usage within Unreal Engine for real-time gameplay development.

###### 

###### \## Systems Included

###### \- Player Character (movement, jumping, input handling)

###### \- Projectile and combat system

###### \- Enemy base class with damage and interaction logic

###### \- Kill volume and respawn behavior

###### 

###### \## Technical Focus

###### \- C++ gameplay system development in Unreal Engine 5

###### \- Object-oriented design for reusable gameplay systems

###### \- Debugging real-time gameplay interactions

###### \- Integration with Unreal Engine components and systems

###### 

###### \## Additional Experience

###### In addition to this C++ work, I have developed a full gameplay vertical slice using Unreal Engine Blueprints, including:

###### \- Enemy AI behaviors (patrol, chase)

###### \- Wave-based encounter systems

###### \- Combat and gameplay loop design

###### 

###### \## Notes

###### This repository is intended as a code showcase and does not include full project assets or Unreal project files.
